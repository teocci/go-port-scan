## go-port-scan

## Introduction
`go-port-scan` package provides simple port scanning examples to learn Go concurrency and goroutine bounding.

## Features
1. simplest
2. sequential
3. too-fast
4. synchronized
5. semaphore
6. semaphore-with-timeout
7. semaphore-with-timeout-per-scan
8. pipeline
9. fan-out-fan-in
10. fan-out-fan-in-with-workers
